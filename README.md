# АС «Агроном» — Мобильное приложение для автоматизации работы агрономов

## Описание проекта
Разработка мобильного приложения для фиксации агротехнических мероприятий, учета удобрений и техники.

## Структура репозитория
- `bin/` — исполняемые файлы приложения (.apk)
- `config/` — файлы настроек
- `docs/` — документация
- `diagrams/` — диаграммы
- `releases/` — архивные версии

## Технологии
- Платформа: 1С:Предприятие 8.3.27
- СУБД: PostgreSQL
